Chemin correspondant au repertoire parent dans lequel se trouve la ressource à deployer. Utile si la ressource à deployer n'est pas générée dans le repertoire workspace de Jenkins. Si absent le workspace sera utilisé comme base.
Attention : Dans le cas d'un deploiement depuis un esclave, il est recommandé de réaliser une copie de l'artifact à deployer dans le workspace et de laisser cette valeur vide.
